# Design System — Stencil Group Vision 2026

## Overview

Este design system define la identidad visual y los componentes reutilizables de la plataforma Vision 2026 de Stencil Group. Basado en el estilo de Perdoo (plataforma OKR): tema claro, profesional, con sidebar oscuro.

## Archivos

| Archivo | Contenido |
|---------|-----------|
| [tokens.md](tokens.md) | Variables CSS: colores, tipografia, espaciado, sombras, radios |
| [components.md](components.md) | Catalogo de componentes: cards, tables, badges, modals, buttons, forms, tabs |
| [layouts.md](layouts.md) | Layouts de pagina: app shell, sidebar, content area, breakpoints |
| [pages.md](pages.md) | Especificaciones por modulo: Dashboard, Gastos, Escenarios |

## Principios de Diseno

1. **Claridad** — Fondo claro, tipografia legible, jerarquia visual clara
2. **Consistencia** — Mismos tokens y componentes en todos los modulos
3. **Densidad informativa** — Tablas y KPIs compactos pero legibles
4. **Accionable** — Botones y acciones visibles, feedback inmediato
